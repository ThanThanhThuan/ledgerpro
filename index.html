LedgerPro - Double Entry Accounting

API đang chạy. Triển khai frontend để sử dụng đầy đủ.

API Endpoints:
  • GET /api/health - Kiểm tra
  • GET /api/accounts - Danh sách tài khoản
  • GET /api/journal-entries - Danh sách phiếu
  • GET /api/reports/balance-sheet - Bảng cân đối
  • GET /api/reports/trial-balance - Bảng thử
